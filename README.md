# CTV Logos

**35 brand logos** for CTV, streaming, and ad tech — black on transparent (SVG or PNG), grouped by category. Shared here so you can grab what you need.

## Get the logos

- **Download everything:** [CTV_Logos.zip](CTV_Logos.zip) — one zip with all logos.
- **Browse by category:** Open the [CTV_Logos](CTV_Logos) folder:
  - **AdTech** — Cineverse, Infillion, Wurl, Frequency, JustWatch, GumGum, Kargo, Canvas Space, Magnite
  - **StreamingPlatforms** — Sling, Prime, Philo, LG Channels, Anoki, Local Now, Fubo, Roku, Plex, Tubi, Tablo, YouTube, TRC, Pluto, DirecTV, TiVo, Freecast
  - **TVPlatforms** — VIZIO, HP, Whale, Vidaa, Hisense, TCL, Rakuten, LG

Logos were generated with the [Brandfetch](https://brandfetch.com) API and converted to black-on-transparent for consistent use in decks and docs.

---

*Want to regenerate or add brands? The script is in `logos/` — you’ll need a free Brandfetch API key. See `logos/README.md` for details.*
