# CTV Logos

**35 brand logos** for CTV, streaming, and ad tech. Each logo comes in **black** and **white** on transparent (SVG or PNG), max 1000px, so they work on light or dark backgrounds and you can tweak opacity as needed.

## Get the logos

- **Download everything:** [CTV_Logos.zip](CTV_Logos.zip) — one zip with all logos.
- **Folder structure:** [CTV_Logos](CTV_Logos) contains:
  - **Black/** — for use on light backgrounds (AdTech, StreamingPlatforms, TVPlatforms)
  - **White/** — for use on dark backgrounds, photos, video (same categories)

Within each, the same brands: AdTech (Cineverse, Infillion, Wurl, Frequency, JustWatch, GumGum, Kargo, Canvas Space, Magnite), StreamingPlatforms (Sling, Prime, Philo, LG Channels, Anoki, Local Now, Fubo, Roku, Plex, Tubi, Tablo, YouTube, TRC, Pluto, DirecTV, TiVo, Freecast), TVPlatforms (VIZIO, HP, Whale, Vidaa, Hisense, TCL, Rakuten, LG).

Generated with the [Brandfetch](https://brandfetch.com) API; solid black/white makes it easy to adjust opacity for more options.

---

*Want to regenerate or add brands? The script is in `logos/` — you’ll need a free Brandfetch API key. See `logos/README.md` for details.*
